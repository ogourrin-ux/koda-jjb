# Site KODA Jiu-Jitsu Academy

Nom de domaine : koda-jjb.com

## Contenu du dossier
- index.html : page d’accueil complète
- styles.css : design responsive sombre / or / blanc cassé
- assets/logo-koda : logo provisoire ou logo récupéré depuis la session

## À compléter avant mise en ligne définitive
1. Remplacer le logo provisoire par le logo final KODA.
2. Ajouter le numéro de téléphone.
3. Compléter le CV martial exact de Yoan Gomez :
   - grade
   - palmarès
   - compétitions nationales/internationales
   - affiliations
   - diplômes d’encadrement
4. Ajouter des photos réelles : Yoan, dojo, cours, kimono/rashguard.
5. Ajouter les mentions légales.
6. Ajouter une politique de confidentialité si formulaire ou analytics.
7. Remplacer le mail par une adresse dédiée type contact@koda-jjb.com.

## Mise en ligne
Déposez à la racine de l’hébergement :
- index.html
- styles.css
- dossier assets
